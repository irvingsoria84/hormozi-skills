# Coloca aquí tus archivos de contexto

Este directorio es para archivos que quieras que la IA lea antes de generar tu estrategia.

Puedes colocar aquí:
- Descripción de tu negocio en texto plano (.txt o .md)
- Tu oferta actual
- El texto de tu página de ventas
- Testimonios de clientes
- Notas de estrategia
- Cualquier documento de referencia

El orquestador leerá automáticamente los archivos en este directorio al inicio de la sesión.
